# Skills

OpenClaw Agent Skills Collection

## Skills List

- **feishu-api** - 向飞书用户发送消息、图片、文件的直接工具
- **home-assistant-mcp** - 通过 MCP Server 控制 Home Assistant 智能家居设备
- **web-search-plus** - 统一搜索技能，智能路由到多个搜索引擎

## Installation

Place skill directories in `~/.openclaw/workspace/skills/` or use ClawHub to install.

## Documentation

Each skill has its own `SKILL.md` with detailed usage instructions.
